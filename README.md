# Проект: Место (Mesto)

Интерактивная страница, на которой можно делиться фотографиями, удалять их и ставить лайки.

## Ссылка на проект
[Посмотреть работающий сайт](https://varvarank.github.io/mesto-production/)

## Функциональность
* **Работа с сервером:** Загрузка начальных карточек и данных профиля, синхронизация лайков и аватара.
* **Управление контентом:** Добавление новых мест и удаление собственных карточек.
* **Валидация:** Формы проверяются на лету (запрет пустых полей, проверка формата ссылок).
* **Инфо-панель:** Просмотр детальной статистики (общее число карточек, уникальные пользователи, даты активности).

## Инструкция по запуску
1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/varvarank/mesto-ad.git](https://github.com/varvarank/mesto-ad.git)
```
2. Установите зависимости:
```bash
npm install
```
3. Запустите проект в режиме разработки:
```bash
npm run dev
```
4. Для сборки проекта (создания папки dist):
```bash
npm run build
```
